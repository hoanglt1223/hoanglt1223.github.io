CSS là ngôn ngữ tạo phong cách cho trang web – Cascading Style Sheet language.

Cascading: (chạy từ trên xuống - dưới đè lên trên)

  • Note: Từ đầu đã có CSS mặc định trong HTML
  1. Nhúng file CSS vào HTML
  • Cách 1: <style> ⇒ có thể dùng cho CSS ngắn but vẫn nên dùng <link> vì dễ quản lý và mở rộng sau này. + Dùng <style> vì có thể dùng cho riêng 1 file HTML hoặc trình duyệt đỡ mất công tải file CSS (TGDĐ dùng <style>) ⇒ tăng tốc độ load trang Trick: đặt tên file vào thẻ head để google ưu tiên hiển thị trên search result.
  • Cách 2: <link> ⇒ có thể dùng 1 file css cho toàn hệ thống, giúp file html gọn gàng sạch sẽ hơn. Nhưng trang web sẽ không thể load hoàn toàn trước khi tải xong file css.
