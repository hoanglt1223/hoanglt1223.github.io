I.

- JavaScript là 1 ngôn ngữ script (kịch bản). Hễ nhắc tới script ⇒ ngôn ngữ bậc cao

- JS ko hề có compiler vì JS dựa trên Interpreter (file đc đọc trực tiếp để cái máy có thể chạy đc)

⇒ ở đây Interpreter là Browser. Ko can thiệp trực tiếp vào hệ thống mà các lớp đã đc lập trình sẵn

- Các ngôn ngữ khác dùng compiler để tạo ra các file mà máy đọc đc. Có thể xoá file gốc vẫn chạy đc.

- JS nếu xoá file thì máy ko chạy đc

- Ngôn ngữ thông dịch (JavaScript): sẽ hướng dẫn cho browser cách tương tác vs OS và Hardware.

- Compiler sẽ nhanh hơn Interpreter vì đc biên dịch thành mã máy và tương tác trực tiếp với OS.

⇒ Điểm yếu: ko bảo đảm là bê qua OS khác sẽ đọc đc.
