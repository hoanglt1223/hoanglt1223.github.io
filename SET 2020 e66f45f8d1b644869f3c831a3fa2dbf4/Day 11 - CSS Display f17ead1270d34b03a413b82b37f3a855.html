  • Display: inline đã có xuất hiện ở 1 số thành phần như <span>

_ đặc tính: 1. ko thể set width, height, margin (trên browser ko chạy, viết vào code ko báo lỗi) ⇒ editor phải cài gì đó để báo _ margin ko đc bao gồm top và bottom

2. element cùng cấp sẽ nằm cùng 1 dòng (ý nghĩa của từ inline)

  • Display: block

_ đặc tính:

  1. có thể set width, height, margin, padding như bth
  1. Một element chỉ nằm trong một dòng: có giảm width hay height gì cũng sẽ bị bù vào bởi margin

  • Inline-block

_ đặc tính:

  1. có thể set width & height
  1. các thuộc tính cùng cấp có thể nằm chung dòng đc

Note: lúc nào cần chung 1 dòng thì xài block ko thì inline-block ⇒ kinh nghiệm thực tế: inline ít đc xài nhất

_ : column (item nằm dọc) (trên xuống dưới)

_ :row_reverse: (phải qua trái)
